# 1.6 Procesamiento de mapas de bits

Más allá de construir los modelos abstractos como en las prácticas trigonométricas anteriores, la disciplina formal del **Procesamiento de Imágenes** (o post-procesamiento digital en tiempo real) se encarga de analizar y alterar la imagen bidimensional como una señal final 2D (Pitas, 2000).

Este procesamiento en la matriz central de píxeles implica operaciones tanto punto a punto como espaciales conjuntas:
- **Ecualización de Histogramas:** Un proceso que aumenta el contraste global en mapas de bits distribuyendo los valores atascados de intensidad de luz de una forma más equitativa, mejorando así dramáticamente fotos sobre o sub-expuestas.
- **Operadores Convulsivos (Kernels):** Se trata de pequeñas matrices (e.j 3x3 o 5x5) que "barren" cada píxel de la foto, reemplazando su valor por la mutua influencia de sus vecinos físicos bajo la matriz matemática. Modificando el contenido del _Kernel_ se consiguen los legendarios filtros de _Blur/Desenfoque_ de paso bajo, y el de _Sharpen_ espacial (Gonzalez & Woods, 2018).

### Ejemplo de Python: Filtro Matriz Convulsiva (Pillow)
En procesamiento de imágenes digitales, el código subyacente de la graficación para desenfocar un raster o mapa de bits implica usar los multiplicadores de Kernel mencionados arriba. La librería `Pillow` maneja estas matrices internas a gran velocidad en la CPU:

```python
from PIL import Image, ImageFilter

# 1. Abrir un mapa de bits original (JPG/PNG como arreglo bidimensional)
imagen_original = Image.open("fotografia.jpg")

# 2. Aplicar función de Kernel (Desenfoque Gaussiano / Interpolación Gausiana)
# Mezcla matemáticamente la luminancia de cada pixel iterativo con el de sus vecinos dictados por el parámetro radius
imagen_desenfocada = imagen_original.filter(ImageFilter.GaussianBlur(radius=5))

# 3. Guardar el resultado procesado como archivo nuevo
imagen_desenfocada.save("resultado_blur.jpg")

print("Se ha difuminado la imagen usando matemáticas de procesamiento espacial convolucional.")
```
